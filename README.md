# Secure Expense Tracker

Full-stack MERN application with JWT authentication and user-based data isolation.

## Live Demo
Frontend: https://expense-tracker-frontend-olive-chi.vercel.app/
Backend: https://expense-tracker-backend-ypt6.onrender.com

## Tech Stack
- React (Vite)
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features
- User signup & login
- Protected routes
- Add / view / delete expenses
- Cloud deployment
